# Post 06 — 모바일 퍼스트 PWA

## 캡션

"앱스토어에 없어요?"
→ 홈 화면에 추가하면 앱이에요 📱

찰칵AI는 PWA(Progressive Web App)입니다.

앱스토어 심사 없이,
다운로드 없이,
브라우저에서 바로 앱처럼 사용.

▪️ 홈 화면 아이콘 추가
▪️ 전체 화면 (standalone) 모드
▪️ 오프라인 지원
▪️ 푸시 알림 (AI 생성 완료 시)

1인 개발자가 iOS + Android 네이티브 앱을
동시에 만드는 건 불가능에 가깝습니다.

PWA로 한 번에 해결 ✅

모바일 퍼스트로 설계해서
375px 작은 화면에서도 완벽하게 동작합니다.
