# Post 03 — 카카오 로그인 구현

## 캡션

한국 서비스에 구글 로그인만 넣으면
절반은 이탈합니다.

찰칵AI는 카카오 로그인을 메인으로 넣었어요 💬

한국 사용자의 90%가 카카오톡을 쓰니까
"카카오로 3초 로그인"이 당연한 선택.

Supabase Auth + 카카오 OAuth 연동은
생각보다 삽질이 좀 있었는데요…

▪️ 카카오 디벨로퍼스 앱 등록
▪️ Redirect URI 설정 (Supabase 콜백)
▪️ 동의항목 세팅 (프로필, 이메일)
▪️ Supabase Provider 설정

한 번 세팅하면 소셜 로그인이
이렇게 편할 수가 없습니다.
