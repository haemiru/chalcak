# Post 07 — 사진 업로드 UX 개선기

## 캡션

사진 업로드 하나에도 UX가 있습니다 📸

처음엔 그냥 파일 선택 버튼 하나였는데
테스트해보니 문제가 산더미.

❌ 모바일에서 카메라 직접 촬영이 안 됨
❌ PC에서 촬영 버튼이 뜨면 혼란
❌ 어떤 사진을 올려야 하는지 모르겠음

그래서 이렇게 바꿨습니다:

✅ 모바일: 직접 촬영 / 앨범 선택 버튼 분리
✅ PC: 파일 업로드만 표시 (촬영 버튼 숨김)
✅ 가이드 텍스트로 좋은 셀카 조건 안내

"사용자가 생각할 필요 없게 만들어라"

개발보다 UX 고민에 시간이 더 걸리는
1인 개발자의 현실입니다 😅
